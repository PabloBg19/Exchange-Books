1.PASO:Descargar la web
Para poder poner la web en visual studio code que es donde se ha creado esta pagina web, debes abrir el visual studio code, y en la parte de arriba a la izquierda en el botón de file le damos a open folder, y una descomprimida nuestra carpeta la pulsamos, decimos que confiamos en los autores y ya tendremos la web y el código listo.

2.PASO:Saber como funciona la Web
La web es bastante sencilla de entender tiene la pagina principal usada para poder ver los libros disponibles, un apartado donde puedes subir tus libros, un blog donde se explica el porque de Exchange books, otra seccion de contacto para poder notificar cualquier problema, un apartado diferenciador de crear historias, otro apartado donde el usuario pueda iniciar sesion o registrarse para poder pedir libros y subirlos y por ultimo un apartado de quiz que le hacen preguntas al usuario para poder conseguir premios.
